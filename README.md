# TaskTraker

Небольшое десктопное приложение на Python/Tkinter для ведения задач с дедлайнами, приоритетами и переключением языка интерфейса (`en`/`ru`).

## Возможности

- добавить задачу с названием, описанием, датой и приоритетом;
- просматривать список задач;
- редактировать и удалять задачи;
- хранить данные локально в SQLite;
- переключать язык интерфейса.

## Установка

```bash
pip install tkcalendar
python main.py
```

## Файлы

- `main.py` — точка входа.
- `task_tracker.py` — интерфейс приложения.
- `database.py` — работа с SQLite.
- `translations.py` — тексты интерфейса.

## Данные

Локальная база `tasks.db` создаётся при запуске и не хранится в Git.
